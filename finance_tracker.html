Cloud Sync (Firebase)

Not connected — data only lives in this browser.
Connect a Firebase project so this tracker's data stays up to date across every device/browser you open it in. Setup instructions.
1. Go to console.firebase.google.com and create a new project (free tier is enough).
2. In the project, go to Build → Firestore Database, click Create database, and start it in test mode.
3. Click the gear icon → Project settings → scroll to "Your apps" → click the </> (web) icon → register an app (no hosting needed) → copy the firebaseConfig object shown.
4. Paste that whole object into the first box above.
5. Make up a long random string (e.g. mash your keyboard, 20+ characters) and paste it into the second box — this is the only thing standing between your data and anyone who has this file, so keep it private and don't reuse a real password.
6. In Firestore, go to the Rules tab and replace the contents with the rule below (swap in your random ID), then click Publish: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /financeTracker/YOUR-RANDOM-ID-HERE {
      allow read, write: if true;
    }
    match /{document=**} {
      allow read, write: if false;
    }
  }
}
7. Click "Connect" above. From then on, this device (and any other device you set up the same way with the same config + ID) will sync automatically.
